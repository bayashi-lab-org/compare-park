**【結論】ノアの高さ制限は全高1,895mm（2WD）/ 1,925mm（4WD）。** 一般的な機械式駐車場の1,550mm制限・1,800mm制限にはどちらも入りません。全高2,000mm以上の大型車対応駐車場が必要です。全幅1,730mm・全長4,695mmは問題なし。ヴォクシーと寸法は同一です。

## ノアの高さ・駐車場サイズ一覧

トヨタ ノアは、ヴォクシーの兄弟車にあたるMクラスミニバンです。ファミリー層に人気が高く、5ナンバーサイズのグレードもあるのが特徴です。

### 90系（現行・2022年〜）の寸法

| 項目 | サイズ |
|------|--------|
| 全長 | 4,695mm |
| 全幅 | 1,730mm |
| 全高 | 1,895mm（2WD）/ 1,925mm（4WD） |
| 重量 | 1,600〜1,710kg |

ノアとヴォクシーは基本的に同じプラットフォーム・同じ寸法です。

## ノアの高さ制限 — 機械式駐車場に入るか？

**ノアは一般的な機械式駐車場にはほぼ入りません。大型車対応（全高2,000mm以上）の駐車場が必要です。**

全高1,895mm（2WD）は普通車サイズの高さ制限（1,550mm）はもちろん、ハイルーフ対応の高さ制限（1,800mm）も超過しています。

### 各高さ制限・サイズ制限での判定

| 制限 | 全長 | 全幅 | 全高 | 重量 | 判定 |
|------|------|------|------|------|------|
| 普通車（1550mm） | OK | OK | **NG** | OK | ❌ NG |
| ハイルーフ（1800mm） | OK | OK | **NG** | OK | ❌ NG |
| 大型車（2000mm） | OK | OK | OK | OK | ✅ OK |

### 2WDと4WDの違い

| 駆動方式 | 全高 | 2,000mm制限 | 余裕 |
|---------|------|-----------|------|
| 2WD | 1,895mm | ✅ OK | 105mm |
| 4WD | 1,925mm | ✅ OK | 75mm |

4WDは全高が30mm高くなりますが、2,000mm制限には収まります。

## Mクラスミニバンとの比較

| 車種 | 全高（2WD） | 全幅 | 2,000mm余裕 |
|------|-----------|------|-----------|
| ステップワゴン | 1,840mm | 1,750mm | 160mm |
| セレナ | 1,870mm | 1,695mm | 130mm |
| **ノア/ヴォクシー** | **1,895mm** | **1,730mm** | **105mm** |
| デリカD:5 | 1,875mm | 1,795mm | 125mm |

ノア/ヴォクシーはMクラスミニバンの中で**全高が最も高い**です。ステップワゴンやセレナの方が全高が低いため、駐車場の選択肢はやや広がります。

## ノアの強み: 全幅が控えめ

全高は高いですが、**全幅1,730mm**はMクラスミニバンの中で標準的で、機械式駐車場の幅制限（1,850mm）に対して**120mmの余裕**があります。全幅がネックになることはまずありません。

## ノアが入る駐車場の条件

- **全高制限**: 1,900mm以上（2WD）/ 1,950mm以上（4WD）
- **全長制限**: 4,700mm以上
- **全幅制限**: 1,750mm以上（余裕あり）
- **重量制限**: 1,800kg以上

## まとめ

- ノアは**全高1,895mm（2WD）/ 1,925mm（4WD）** のため、ハイルーフ対応でも入らない
- **大型車対応（全高2,000mm以上）** の機械式駐車場が必要
- ヴォクシーと寸法は同一なので、駐車場適合も同じ
- Mクラスでは**ステップワゴン（1,840mm）やセレナ（1,870mm）** の方が全高が低い
- 全幅1,730mmは問題なし
