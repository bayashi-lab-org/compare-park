**【結論】レクサスNXは全幅1,865mmが一般的な幅制限（1,850mm）を15mm超過するため、幅1,900mm以上対応の駐車場が必要。** 全高1,660mmはハイルーフ対応で余裕がありますが、全幅がネックとなり一般的な機械式駐車場には入りません。NX450h+（PHEV）は重量1,990kgで重量制限にも要注意です。

## レクサスNXの寸法スペック

レクサス NXは、レクサスのコンパクトSUVです。ハリアーと同じプラットフォーム（GA-K）をベースとしており、サイズ感も近いですが、全幅がやや広い点に注意が必要です。

### 2代目（現行・2021年〜）の寸法

| グレード | 全長 | 全幅 | 全高 | 重量 |
|---------|------|------|------|------|
| NX250 2WD | 4,660mm | 1,865mm | 1,660mm | 1,620kg |
| NX250 AWD | 4,660mm | 1,865mm | 1,660mm | 1,680kg |
| NX350h 2WD | 4,660mm | 1,865mm | 1,660mm | 1,720kg |
| NX350h AWD | 4,660mm | 1,865mm | 1,660mm | 1,790kg |
| NX350 AWD | 4,660mm | 1,865mm | 1,660mm | 1,740kg |
| NX450h+ PHEV AWD | 4,660mm | 1,865mm | 1,660mm | 1,990kg |

全グレードで寸法は同一（全長4,660mm・全幅1,865mm・全高1,660mm）。違いは**重量のみ**です。

## 高さ制限との比較 — 機械式駐車場に入るか？

**レクサスNXはハイルーフ対応の機械式駐車場なら全高は問題なし。ただし全幅1,865mmが一般的な幅制限（1,850mm）を15mm超過します。**

### 各高さ制限・サイズ制限での判定

| 制限 | 全長 | 全幅 | 全高 | 重量 | 判定 |
|------|------|------|------|------|------|
| 普通車（1550mm） | OK | **NG** | **NG** | OK | ❌ NG |
| ハイルーフ（1800mm/幅1850mm） | OK | **NG** | OK | OK | ❌ NG（全幅超過） |
| ハイルーフ（1800mm/幅1900mm） | OK | OK | OK | OK | ✅ OK |
| 大型車（2000mm） | OK | OK | OK | OK | ✅ OK |

### 全幅1,865mmの問題

NXの全幅1,865mmは以下の制限との関係になります。

| 幅制限 | 判定 | 備考 |
|--------|------|------|
| 1,850mm | ❌ NG | 15mm超過 |
| 1,900mm | ✅ OK | 35mmの余裕 |
| 1,950mm | ✅ OK | 85mmの余裕 |

**全幅1,900mm以上対応**の駐車場を選ぶ必要があります。

### NX450h+（PHEV）の重量問題

NX450h+はPHEV（プラグインハイブリッド）で、車両重量が**1,990kg**と2,000kgに迫ります。

| グレード | 重量 | 2,000kg制限 |
|---------|------|-----------|
| NX250 | 1,620〜1,680kg | ✅ 余裕あり |
| NX350h | 1,720〜1,790kg | ✅ OK |
| NX350 | 1,740kg | ✅ OK |
| NX450h+ | 1,990kg | ⚠️ ギリギリ |

NX450h+は重量制限2,000kgの駐車場でギリギリ。**重量制限2,100kg以上**の駐車場を選ぶのが安心です。

## ハリアーとの比較

| 項目 | レクサス NX | ハリアー |
|------|-----------|---------|
| 全長 | 4,660mm | 4,740mm |
| 全幅 | 1,865mm | 1,855mm |
| 全高 | 1,660mm | 1,660mm |
| 重量 | 1,620〜1,990kg | 1,530〜1,700kg |
| 幅1,850mm制限 | ❌ NG（15mm超過） | ❌ NG（5mm超過） |

NXはハリアーより全幅が10mm広いため、駐車場の選択肢がわずかに狭くなります。どちらも**全幅1,900mm以上**の駐車場が必要です。

## レクサスNXが入る駐車場の条件

- **全高制限**: 1,700mm以上
- **全幅制限**: 1,900mm以上（必須）
- **全長制限**: 4,700mm以上
- **重量制限**: 1,800kg以上（NX450h+は2,100kg以上推奨）

- レクサスNXは**全幅1,865mm**のため、一般的な機械式駐車場（幅1,850mm制限）には入らない
