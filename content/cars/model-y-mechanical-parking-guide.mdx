**【結論】テスラ Model Yは多くの機械式駐車場に入りません。** 全幅1,921mmが一般的な幅制限1,850mmを大幅に超過。さらに車両重量1,930〜1,979kgは重量制限2,000kgギリギリです。全高1,624mmはハイルーフ対応なら問題ありませんが、幅と重量がネックになります。

## テスラ Model Yの寸法スペック

テスラ Model Yは、テスラのベストセラーモデルであるコンパクトSUVタイプのEV（電気自動車）です。EVならではのバッテリー重量による機械式駐車場の**重量制限**に注意が必要です。

### 現行モデルの寸法

| 項目 | サイズ |
|------|--------|
| 全長 | 4,751mm |
| 全幅 | 1,921mm |
| 全高 | 1,624mm |
| 重量 | 1,930kg（RWD）/ 1,979kg（Long Range AWD） |

## 高さ制限・重量制限との比較 — 機械式駐車場に入るか？

**Model Yは全幅と重量がネックで、多くの機械式駐車場に入りません。**

全高1,624mmは普通車サイズ（1,550mm）はNGですがハイルーフ対応なら問題ありません。しかし全幅1,921mmは一般的な機械式駐車場の制限を超えています。

### 各高さ制限・サイズ制限での判定

| 制限 | 全長 | 全幅 | 全高 | 重量 | 判定 |
|------|------|------|------|------|------|
| 普通車（1550mm） | OK | **NG** | **NG** | **NG** | ❌ NG |
| ハイルーフ（1800mm） | OK | **NG** | OK | **NG** | ❌ NG |
| 大型車（2000mm） | OK | OK | OK | OK | ✅ OK |

### 全幅1,921mmの問題

Model Yの全幅1,921mmは以下の制限を超えます。

- 普通車: 1,850mm → **NG**（71mm超過）
- ワイド対応: 1,900mm → **NG**（21mm超過）
- 大型対応: 1,950mm → **OK**

全幅1,950mm以上対応の駐車場が必要です。

### EVの重量問題

EVはバッテリーの重さにより、同サイズのガソリン車より**200〜500kg重い**のが特徴です。

| 車種 | 重量 | 同サイズガソリン車との差 |
|------|------|----------------------|
| Model Y RWD | 1,930kg | +300〜400kg |
| Model Y Long Range | 1,979kg | +350〜450kg |

一般的な機械式駐車場の重量制限は1,600〜2,000kgです。Model Y RWD（1,930kg）は2,000kg制限の駐車場であればギリギリ収まりますが、余裕はほとんどありません。

## Model Yが入る駐車場の条件

- **全高制限**: 1,650mm以上
- **全幅制限**: 1,950mm以上
- **全長制限**: 4,800mm以上
- **重量制限**: 2,000kg以上（余裕を考えると2,100kg以上が安心）
