**【結論】タントはハイルーフ対応の機械式駐車場なら2WD・4WDとも入る。** 全高1,755mm（2WD）/1,775mm（4WD）はハイルーフ制限（1,800mm）に対して25〜45mmの余裕があります。N-BOXの4WD（1,815mm）が1,800mm制限を超過するのに対し、タントは4WDでも収まるため、スーパーハイトワゴンで最も機械式駐車場に入りやすい車種です。

## タントの寸法スペック

ダイハツ タントは、N-BOXと並ぶ人気のスーパーハイトワゴン軽自動車です。ミラクルオープンドア（助手席側の大開口）が特徴で、子育て世帯を中心に高い支持を得ています。

### 4代目 LA650S/LA660S（現行・2019年〜）の寸法

| 項目 | サイズ |
|------|--------|
| 全長 | 3,395mm |
| 全幅 | 1,475mm |
| 全高 | 1,755mm（2WD）/ 1,775mm（4WD） |
| 重量 | 880〜960kg |

## 高さ制限との比較 — 機械式駐車場に入るか？

**タントはハイルーフ対応の機械式駐車場であれば、2WD・4WDとも入ります。**

全長・全幅は軽自動車規格なので全く問題ありません。全高が1,755〜1,775mmなので、ハイルーフ対応（1,800mm）に対して25〜45mmの余裕があります。

### 各高さ制限・サイズ制限での判定

| 制限 | 全長 | 全幅 | 全高 | 重量 | 判定 |
|------|------|------|------|------|------|
| 普通車（1550mm） | OK | OK | **NG** | OK | ❌ NG |
| ハイルーフ（1800mm） | OK | OK | OK | OK | ✅ OK |
| 大型車（2000mm） | OK | OK | OK | OK | ✅ OK |

### 2WDと4WDの違い

| 駆動方式 | 全高 | 1,800mm制限 | 余裕 |
|---------|------|-----------|------|
| 2WD | 1,755mm | ✅ OK | 45mm |
| 4WD | 1,775mm | ✅ OK | 25mm |

N-BOXとの最大の違いはここです。**タントは4WDでも1,800mm制限に収まります**が、N-BOXの4WDは1,815mmで入りません。

## N-BOXとの比較

| 項目 | タント | N-BOX |
|------|--------|-------|
| 全高（2WD） | 1,755mm | 1,790mm |
| 全高（4WD） | 1,775mm | 1,815mm |
| 1,800mm制限（2WD） | ✅ OK（余裕45mm） | ✅ OK（余裕10mm） |
| 1,800mm制限（4WD） | ✅ OK（余裕25mm） | ❌ NG（15mm超過） |
| 重量 | 880〜960kg | 910〜990kg |

**機械式駐車場の利用を考えるなら、タントはN-BOXより有利**です。特に4WDモデルでは明確な差があります。

## スーパーハイトワゴン全体の比較

| 車種 | 全高（2WD） | 全高（4WD） | 1,800mm余裕（4WD） |
|------|-----------|-----------|-----------------|

タントはスーパーハイトワゴンの中で**全高が最も低い**車種です。

## タントが入る駐車場の条件

- **全高制限**: 1,800mm以上であればOK（2WD/4WDとも）
- **全長・全幅・重量**: 軽自動車規格なので全ての機械式駐車場で問題なし
